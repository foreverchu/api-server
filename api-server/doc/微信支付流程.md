微信支付流程:
====

1. 申请好apkey与apisecret
2. mch第三方Pakcage


实际流程:
----

1. 根据接口拼接好参数(其中一个是回调地址), 通过mch发送到微信服务器, 返回一个支付地址
2. 根据支付地址, 生成二维友一, 用户扫描后, 在微信端支付
3. 支付结果会通过第一步传过去的回高地址接收到处理


具体在玩比赛官网流程:
----

1. 用户在报名成功后生成一张订单
2. 用户选择支付方式
3. wanbisai发送一个请求, 传入参数: OrderId, GameId, PlayerId,PayType, 发送到api-server, 其中回调地址应该通过参数传给api-server
> 因为不同的支付可能需要不一样, api-server会分别针对每一个支付做一个接口

4. api-server处理此请求, 向微信服务器发送请求, 接收到个支付地址, 将此地址返回给wanbisai
5. wanbisai接收支付地址, 并且生成一张二维码图片, 供用户扫描
6. 微信服务器处理完成后调用回调地址, 玩比赛此结果发送给api-server
7. api-server更新数据, 将结果返回给玩比赛
8. 玩比赛更新用户界面
